\section{What Would Change This View}

\begin{itemize}
  \item The full stack---object, bundle, bearer, correction, successor, selection---reads green on every measurable proxy while the outcome is catastrophic, because at the superintelligent limit each proxy is something the system can \emph{present} rather than something we can verify (Chapter~\ref{ch:verifiability-ontology}).
  \item The book's master disconfirmer: exhibit a single core metric that is \emph{adversarially} verifiable---not merely observable, not merely i.i.d.-robust---against a system optimizing against it. If none can be exhibited, every chapter inherits the verification gap.
\end{itemize}
