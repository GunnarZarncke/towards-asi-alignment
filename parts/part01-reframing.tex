\part{The Alignment Problem Reframed}

This part changes the object of the alignment problem. It starts from the possibility that the relevant target is not the visible model, not a fixed utility function, and not a one-time behavioral property, but a human-correctable value-bearing process embedded in civilization. The chapters define the scope, assumptions, and failure coverage the rest of the book will rely on.
