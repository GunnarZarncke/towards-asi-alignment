# Chapter 16 — Value bundle simulator

Toy model: map ecological/social environment parameters to culturally salient
value terms (Schwartz-like and extended labels).

**Not empirical.** Weights are editable priors for hypothesis generation, aligned
with the value-bundle framing in `chapters/ch16-value-bundle-model.tex`.

Run via `python3 serve.py` from `src/`, then open
[demos/ch16-value-bundle-simulator/](http://127.0.0.1:8765/demos/ch16-value-bundle-simulator/).
